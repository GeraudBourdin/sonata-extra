Installer l’extension initiale : https://chrome.google.com/webstore/detail/videostream-for-google-ch/cnciopoikihiagdjbjpnocolokfelagl?hl=en

  1. Aller dans => chrome://flags/

  2. rechercher « Views Cast dialog »

  3. dans le parametre « Views Cast dialog », changer « Default » vers « Disabled »

  4. rechercher « Upcoming UI feaures »

  5. dans le parametre « Upcoming UI features » f, changer « Default » vers « Disabled »

  6. Relancer chrome. la popup de recherche de chrome cast devrait ensuite apparaitre.
