Pour activer le lecteur de carte sd sur un portable sous ubuntu :
1 – Backuper votre fichier de module :
sudo cp /etc/modules /etc/modules.bak

2 – Editez le fichier module et ajoutez le module « tifm_sd » a la fin du fichier
sudo gedit /etc/modules or sudo vi /etc/modules

3 – Redémarrez la machine ou activer modules :
sudo modprobe tifm_sd
