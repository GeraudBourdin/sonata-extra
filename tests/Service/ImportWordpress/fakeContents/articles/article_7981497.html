Tous ces petits outils seront utiles pour les paranoïaques, mais dans mon cas, il font planter la machine et me rendent improductif.
Après les avoir supprimé, la machine est devenue enfin stable.

1 – Désactiver les rapport d’erreurs.
vi /etc/default/apport

mettre le paramètre :
enabled=0

2 – Supprimer snapd

sudo apt autoremove --purge snapd
sudo apt-get remove --purge gnome-software-plugin-snap
rm -fr ~/snap

3 – Supprimer apparmor
sudo /etc/init.d/apparmor stop
sudo update-rc.d -f apparmor remove

4 – installer un kernel recent
sudo apt-get install --install-recommends linux-generic-hwe-18.04 xserver-xorg-hwe-18.04
