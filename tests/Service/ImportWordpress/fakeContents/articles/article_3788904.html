Pour que Zend IDE se lance plus rapidement, il suffit de désactiver tous les modules initialisé au lancement.
Window -> Préférences -> General -> Startup and Shutdown
Et décochez tout

Pour que Zend IDE n’utilise plus Git Staging, il suffit de le supprimer au lancement par défaut.
Window -> Préférences -> Team -> Git -> Commiting
Et décochez la première case « Use Staging View to commit instead of Commit Dialog »
