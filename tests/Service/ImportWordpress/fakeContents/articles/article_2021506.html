Lancer le gestionnaire de préférences d’applications au démarrage :

gnome-session-properties

Puis ajouter un item viber avec la commande suivante :

Name: Viber
Command: /opt/viber/Viber StartMinimized
Comment: Commentaire par défaut.

Pour lancer skype avec un user spécifique (pour le lancé minimisé, options->Démarrer Skype minimisé dans la barre d’état système):

Name: Skype
Command: /bin/sh -c  "echo 'tbourdin.partitech xxxxx' | skype --pipelogin"
Comment: Skype.
