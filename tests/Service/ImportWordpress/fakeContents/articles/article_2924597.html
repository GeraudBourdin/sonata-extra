Pour installer skype, il est preferable d’utiliser le repo d’ubuntu, qui va installer tout ce qui va bien pour gérer le son et la video.
En prenant le package disponible sur le site officiel, il est probable que le son ne soit pas géré convenablement.

1 – Installer skype :
sudo add-apt-repository -y ppa:andykimpe/skype && sudo apt-get update
sudo apt-get install skype

2 – Installer le wrapper :
sudo add-apt-repository ppa:skype-wrapper/ppa
sudo apt-get update && sudo apt-get install skype-wrapper
