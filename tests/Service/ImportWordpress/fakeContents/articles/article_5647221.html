Information récupéré a l’adresse : http://tomtomtom.org/networkmanager-openconnect/

Ça a marché du premier coup sur ma machine.

On supprime les paquets inutilisables :

sudo apt-get purge network-manager-openconnect network-manager-openconnect-gnome

On compile les dépendences :

sudo apt-get build-dep network-manager-openconnect

Les sources dans apt doivent etre actives :

sudo sed -i s/#deb-src/deb-src/g /etc/apt/sources.list

On installe les nouvelles dépendances :

sudo apt-get install libnm-dev

On récupere le code source :

wget http://ftp.gnome.org/pub/GNOME/sources/NetworkManager-openconnect/1.1/NetworkManager-openconnect-1.1.93.tar.xz

On se place dans le répertoire :

cd NetworkManager-openconnect-1.1.93

On lance le configure :

./configure --prefix=/usr --sysconfdir=/etc --localstatedir=/var --libexecdir=/usr/lib/NetworkManager/ --enable-more-warnings=yes --disable-static

On compile:

make

On installe :

sudo make install

On install openconnect :

sudo apt-get install openconnect

On compile le package avec checkinstall :

sudo checkinstall

On ajoute un utilisateur systeme :

sudo adduser --system --quiet --home /var/lib/NetworkManager --no-create-home --gecos "NetworkManager OpenConnect plugin" --group nm-openconnect}

Et on redémarre.
